## METERS

TENSION (1-10): Narrative conflict and stakes
- Low (1-3): Calm, relaxed, low stakes
- Medium (4-6): Uneasy, conflict brewing
- High (7-8): Danger imminent, pressure mounting
- Critical (9-10): Breaking point, now or never

HEAT (1-5): Romantic/intimate intensity
- 1: Professional distance
- 2: Subtle awareness
- 3: Flirtation, meaningful touches
- 4: Passion restrained
- 5: Unbridled desire

## BEATS

- exposition: Setting scene, establishing context
- confrontation: Direct conflict between characters
- revelation: Secrets revealed, information changes everything
- intimate-moment: Vulnerable or romantic connection
- climax: Peak moment, everything comes to a head
- resolution: Conflict resolves, new equilibrium

## DIRECTOR'S NOTES

If a "Director's Note" section is provided in the context:
- The director gave this instruction to the agent BEFORE they spoke
- Evaluate whether the agent's dialogue follows this direction
- Set `director_aligned: true` if the agent followed the note
- Set `director_aligned: false` if the agent ignored or contradicted it

If no Director's Note is provided:
- Set `director_aligned: true` (no direction to follow)
